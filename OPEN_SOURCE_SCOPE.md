# 純程式碼公開範圍

## 已包含

- FastAPI 後端、SQLite 結構與 API 原始碼。
- 桌面版與手機版 HTML、CSS、JavaScript。
- 啟動、停止、重啟、安裝與一般維護腳本。
- 遊戲目錄解析、同步預覽、資料治理、公告、兌換碼與帳號管理程式碼。
- 空白資料模式及公開版檢查工具。

## 明確排除

- 所有 PNG、JPG、JPEG、WebP、GIF、ICO、SVG、BMP、AVIF 圖片。
- favicon、專案代表圖、遊戲圖示、Threads、Discord、GitHub 品牌圖示。
- 正式成就名稱、條件、獎勵、版本、官方 ID 清單與內嵌補全記錄。
- `data/catalogs`、`data/relations`、`data/sources` 的正式內容。
- SQLite 資料庫、帳號、工作階段、操作紀錄、郵件、匯入紀錄、日誌與備份。
- `.env`、SMTP 密碼、網域驗證檔與正式站台私人設定。
- 私有發布包的更新日誌、完整維護手冊、發布清單及內部雜湊檔。

## 空白啟動模式

公開版 `.env.example` 預設 `OPEN_SOURCE_EMPTY_DATA=1`。此模式只略過需要正式遊戲資料的同步、筆數門檻與官方 ID 驗證；資料表、帳號、公告、兌換碼及管理功能仍會初始化。

管理員日後可提供自有或獲授權的資料。將資料加入公開儲存庫前，應自行完成著作權、資料庫權及來源條款審查。
